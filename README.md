Butterfly is a real-time inspection intelligence app built for CAT daily walkarounds.

It transforms a technician’s phone into a hands-free, multimodal inspection engine, fusing live video, voice, and machine telemetry into structured, evidence-backed CAT reports.

Instead of filling out forms after an inspection, technicians simply walk the machine. Butterfly streams input through a multimodal AI pipeline, auto-completes the official inspection schema, attaches supporting evidence, and generates a Cat-format report instantly.

Designed for the CAT 320E, Butterfly is architected to scale across machines and integrate directly with VisionLink/Product Link data.
